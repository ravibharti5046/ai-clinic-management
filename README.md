# AI Clinical Operations Platform

Enterprise-grade hospital and clinic ERP system with AI-powered clinical operations support.

## Tech Stack

- Backend: Django + Django REST Framework
- Frontend: React.js
- Database: PostgreSQL
- Authentication: JWT
- AI: OpenAI API / Local LLM
- Messaging: WhatsApp API
- Payments: Razorpay / Stripe
- Deployment: Docker + Nginx + VPS

## Core Modules

- Patient Management
- Doctor Management
- Appointment System
- Billing & Insurance
- Pharmacy Management
- Lab Management
- AI Doctor Assistant
- AI Receptionist
- AI Report Reader
- AI Queue Prediction
- AI Follow-Up Agent

## Safety Disclaimer

AI features are for clinical decision support only. Final medical decisions must be reviewed and approved by a licensed doctor.
